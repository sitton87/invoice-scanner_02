סיכום תהליך Image Mode

קריאות API לקלוד:

  • API #1 (INTRO): תמונה + פרומפט פרטי חשבונית → JSON עם פרטים כלליים
  • API #2 (MAIN): תמונה + פרומפט רשימת פריטים → JSON עם שורות פריטים

סקריפטים מעורבים:

  • ui.py - ממשק משתמש ותיאום
  • full_processor.py - מנהל מרכזי ואיחוד תוצאות
  • intro_analyzer.py - עיבוד פרטי חשבונית (INTRO בלבד)
  • processor.py - עיבוד רשימת פריטים (MAIN בלבד)

עיבוד קבצים מיוחד:

  • PDF: פתיחה עם PyMuPDF → המרת עמודים → איחוד אנכי → אופטימיזציה
  • תמונות: בדיקת גודל → המרה ל-RGB → שינוי גודל → דחיסה
